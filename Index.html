Git e GitHub

Comandos Git:


Comando Descrição
git init Marca a pasta onde será o repositório do git
git status Mostra o status do git
git config --local user.name "Seu nome aqui"
git config --local user.email "seu@email.aqui"		 Configuração de usuário
git add ./
git add "nome-arquivo" 		Adiciona os arquivos para o git, podendo ser um ou todos do repositório
git commit -m "Comentário commit" Salvas as informações das alterações
git log
git log --oneline git log -p 		Mostrar os logs, mais comandos em:
https://devhints.io/git-log
git remote add origin "Path Online" Configurar um caminho online
git push origin master Envia as requisições para o repositório
git branch branch-name Cria um novo branch
git checkout branch-name Começa a trabalhar no novo branch
git merge branch-name Realiza o merge entre branchs
git rebase branch-name Juntar os commits
git checkout -- nome-arquivo Cancelas as alterações que não foi feito o commit, voltando as últimas alterações
git reset HEAD nome-arquivo Cancela as alterações que foram adicionas para commit
git revert hash-commit Cancelar um commit específico
git stash Salvas as alterações para depois
git stash list Mostra as alterações guardadas
git stash pop Retira da lista de modificações e aplica
